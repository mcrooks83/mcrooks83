🚀 Nexus N3
Nexus N3 is a modular engineering toolchain for building sensor-driven research and operational systems.
It brings together sensors, embedded hardware, edge software, algorithms and data workflows, supporting systems that need to operate locally, reliably and independently of continuous cloud connectivity.
